Org-standard. Severity set by technical teams; Priority derived from Severity × Likelihood × Urgency.

📍 Sites

🤝 Operator Partners

🏖️ Holidays

Add holidays to exclude them from schedule duration calculations. Tasks will not count these days when computing timelines.

🔗 Jira Integration

Set your Jira base URL so ticket IDs in Issues and Scope become clickable links.

Example: ticket MAY-1234 opens yourcompany.atlassian.net/browse/MAY-1234
